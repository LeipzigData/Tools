Leipzig Data Event Widget

Mit den Leipzig Data Event Widgets werden die prinzipiellen Möglichkeiten des Leipzig Data Events Frameworks demonstriert.
Sie können die einzelnen Schritte des Workflows bis hin zur Präsentation mit dem hier ausgewählten Thema selbst nachvollziehen:
  • Lokales json-Objekt erzeugen (Speichern der Datei erfordert Schreibrecht des Webservers im Verzeichnis. Alternativ kann das json-Objekt von der Konsole aus erzeugt werden)
  • Präsentation anzeigen
Die Präsentation ist Teil des Leipzig Open Data Projekts.
