Leipzig Data Event Widget

Mit den Leipzig Data Event Widgets werden die prinzipiellen Möglichkeiten des Leipzig Data Events Frameworks demonstriert.
Für diese Präsentation ist die wöchentliche Aktualisierung bereits als Datei EventsDump.ttl eingespielt und im Store aktualisiert. Sie können nun die einzelnen Schritte des Workflows bis hin zur Präsentation mit dem hier ausgewählten Thema selbst nachvollziehen:
  • Store-Daten aktualisieren
  • Lokales json-Objekt erzeugen
  • Zur Präsentationsseite
  • Technische Dokumentation
Die Präsentation ist Teil des Leipzig Open Data Projekts.
